Tutorial Hydra di Termux

Berikut adalah panduan lengkap untuk menggunakan Hydra di Termux:

  • 1. Instalasi Hydra

    Untuk menginstal Hydra di Termux, jalankan perintah berikut:

    pkg update && pkg upgrade
pkg install hydra

  • 2. Pengenalan Hydra

    Hydra adalah alat untuk melakukan brute-force pada berbagai protokol seperti SSH, FTP, HTTP, dan lainnya.

    Catatan: Gunakan alat ini secara bertanggung jawab.

  • 3. Contoh Penggunaan Hydra

    Berikut adalah contoh penggunaan Hydra pada layanan SSH:

    hydra -l username -P passwordlist.txt ssh://target_ip

    Keterangan:

    • -l: Username target
    • -P: Path ke file berisi daftar password
    • ssh://: Protokol yang digunakan
    • target_ip: IP atau domain target

  • 4. Contoh Brute-Force Login FTP

    Untuk brute-force login FTP, gunakan perintah berikut:

    hydra -l admin -P passwordlist.txt ftp://target_ip

  • 5. Cara Membuat Password List

    Anda bisa membuat daftar password sederhana dengan perintah berikut:

    echo -e "password1\npassword2\npassword3" > passwordlist.txt

  • 6. Tips Menggunakan Hydra

    Beberapa tips saat menggunakan Hydra:

    • Gunakan daftar password yang relevan.
    • Pastikan target memiliki izin untuk diuji.
    • Jangan gunakan alat ini untuk aktivitas ilegal.
