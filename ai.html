Kumpulan Tools Termux

Berikut adalah daftar 10 tools Termux yang berguna beserta panduan penggunaannya:

  • 1. Metasploit

    Kegunaan: Framework untuk pengujian keamanan jaringan.

    Cara Menggunakan: pkg install metasploit, kemudian jalankan dengan msfconsole.

  • 2. Nmap

    Kegunaan: Pemindai jaringan untuk mendeteksi port terbuka.

    Cara Menggunakan: pkg install nmap, lalu gunakan nmap [IP].

  • 3. Hydra

    Kegunaan: Alat brute force untuk login ke server tertentu.

    Cara Menggunakan: pkg install hydra, lalu jalankan dengan hydra -l user -P passlist.txt [target].

  • 4. Sqlmap

    Kegunaan: Mendeteksi dan mengeksploitasi SQL Injection.

    Cara Menggunakan: pkg install sqlmap, gunakan sqlmap -u [URL].

  • 5. Node.js

    Kegunaan: Menjalankan JavaScript di server.

    Cara Menggunakan: pkg install nodejs, lalu jalankan file JS dengan node [file.js].

  • 6. Wget

    Kegunaan: Mengunduh file dari internet.

    Cara Menggunakan: pkg install wget, lalu gunakan wget [URL].

  • 7. Python

    Kegunaan: Menjalankan atau membuat skrip Python.

    Cara Menggunakan: pkg install python, lalu jalankan dengan python [file.py].

  • 8. Termux API

    Kegunaan: Mengakses fitur perangkat seperti GPS, SMS, dan lain-lain.

    Cara Menggunakan: pkg install termux-api, jalankan seperti termux-battery-status.

  • 9. Git

    Kegunaan: Mengelola kode dan mengunduh proyek dari GitHub.

    Cara Menggunakan: pkg install git, gunakan git clone [URL].

  • 10. Neofetch

    Kegunaan: Menampilkan informasi sistem dengan grafis.

    Cara Menggunakan: pkg install neofetch, lalu jalankan dengan neofetch.
