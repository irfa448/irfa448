Panduan SQL

Cara Install dan Menggunakan SQL di Termux dengan Mudah

Pengenalan SQL

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengakses, mengelola, dan memanipulasi data dalam database. SQL banyak digunakan dalam aplikasi yang membutuhkan interaksi dengan database.

Cara Install SQL di Termux

Ikuti langkah-langkah berikut untuk menginstall SQL di Termux:

pkg update && pkg upgrade
pkg install mariadb
mysql_install_db
mysqld_safe --user=mysql &

Perintah di atas akan menginstall MariaDB, sebuah software database yang mendukung SQL. Setelah proses selesai, jalankan perintah mysql -u root -p untuk masuk ke dalam konsol MariaDB.

Cara Menggunakan SQL

Berikut adalah beberapa contoh perintah SQL dasar:

  • Membuat Database:
    • CREATE DATABASE nama_database;
  • Membuat Tabel:
    • CREATE TABLE nama_tabel (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(255) NOT NULL,
    umur INT NOT NULL
);
  • Menambahkan Data:
    • INSERT INTO nama_tabel (nama, umur) VALUES ('Ali', 25);
  • Menampilkan Data:
    • SELECT * FROM nama_tabel;
  • Menghapus Data:
    • DELETE FROM nama_tabel WHERE id=1;

Tips dan Catatan

Gunakan SQL dengan bijak untuk mengelola data Anda. Hindari kesalahan seperti menghapus tabel tanpa cadangan (backup).
