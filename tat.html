Panduan Tools GitHub untuk Hacking

1. Pish Web Tool

Tool ini digunakan untuk membuat phishing website. Phishing adalah teknik untuk mendapatkan informasi sensitif pengguna seperti username dan password.

Perintah:

git clone https://github.com/Cabdulahi/pish

Cara Menggunakan:

Setelah di-clone, masuk ke folder pish dan jalankan python3 tool.py. Ikuti langkah-langkahnya sesuai petunjuk di layar.

2. MITM Attacker Tool

Tool ini berguna untuk melakukan serangan Man-in-the-Middle (MITM). Cocok untuk simulasi keamanan jaringan.

Perintah:

git clone https://github.com/websploit/websploit

Cara Menggunakan:

Masuk ke folder websploit, lalu jalankan python3 websploit.py. Tool ini memiliki banyak fitur yang bisa digunakan untuk eksplorasi jaringan.

3. Killshot - Pentesting Framework

Framework all-in-one untuk penetration testing. Cocok untuk mengevaluasi kerentanan sistem secara komprehensif.

Perintah:

git clone https://github.com/bahaabdelwahed/killshot

Cara Menggunakan:

Ikuti dokumentasi pada repositori GitHub untuk instalasi dan pengoperasian framework ini.

4. OSIF - Facebook Information Gathering

Tool ini digunakan untuk mengumpulkan informasi akun Facebook menggunakan API Facebook.

Perintah:

git clone https://github.com/CiKu370/OSIF.git

Cara Menggunakan:

Masuk ke folder OSIF, jalankan python3 osif.py, dan login menggunakan token Facebook Anda.

5. Facebash - Facebook Brute Force

Tool brute force attack untuk Facebook yang menggunakan TOR untuk anonimitas.

Perintah:

git clone https://github.com/thelinuxchoice/facebash

Cara Menggunakan:

Masuk ke folder facebash, lalu jalankan bash facebash.sh. Pastikan TOR sudah terinstal di sistem Anda.
